# chat-system-server

> An thinkjs project

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8360
npm start

# build electron application for production
npm run build

# run unit tests
npm test


# lint all JS/Vue component files in `src/`
npm run lint

```
